
# Alumni (Front-end)

Project for showing all alumni of codecamp




## Clone the project

```bash
  git clone https://github.com/CC-DEVMODE/user-fe-next
```

Go to the project directory

```bash
  cd user-fe-next
```



## Installation

Install my-project with makefile

```bash
  make setup
```

## Running Project

```bash
   pnpm run dev
```
## Tech Stack

Tech stack used in this project

### Framework

* Next.js

### CSS

* Tailwind

### UI Components

* Shadcn
